Detective story: Munhunt for killer. Version 1.0. 22.02.2021

Краткое описание программы:
Данное приложение развивает логику пользователя путем решения головоломок и раскрытия преступления. Собирайте улики, общайтесь со свидетелями, чтобы найти убийцу.

Минимальные требования:
Наличие компьютера
Наличие игровой мыши и клавиатуры
Наличие свободного места на ПК

Инструкция по установке:
1. Скачайте программу
2. Установите
3. Программа работает только на ОС Windows
4. Если игра не запускается, убедитесь, что ваша версия является последней
5. Игра может долго запускаться в зависимости от мощности вашего компьютера. Пожалуйста, будьте терпимее и подождите.
6. Если игра зависает/вылетает/не работает, обратитесь в службу поддержки для решения проблемы.
